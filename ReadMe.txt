To run the task:
1- Use DatabaseBackup to restore and create database schema that store the tables and transactions.
2- Database configuration such as username and password are located in config/database.php you can change their values as your mysql database username and password values (by default my username = root and password = root).
3- Place the project to server folder such as xampp.
4- Run the project via any browser via this link: http://localhost/nqraa-task-master/views/projectInvoice.php
